# Oil Price & Geopolitical Risk Analysis (1990–2025)

This project is an end-to-end Power BI dashboard analyzing how geopolitical events and Iran’s oil exports impact global oil prices.

---

## Tools Used
- Power BI
- DAX
- Power Query

---

## Dashboard Pages
- Overview Dashboard
- Events Analysis
- Exports Analysis

---

## Key Insights
- Oil prices rise during geopolitical instability  
- Sanctions significantly impact exports and revenue  
- High-severity events drive price volatility  

---

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (CSV files)
